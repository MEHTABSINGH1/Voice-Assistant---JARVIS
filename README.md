# 🤖 JARVIS Voice Assistant

An intelligent **Python-based Voice Assistant** inspired by **JARVIS (Iron Man)** that can listen to voice commands and perform tasks automatically like telling time, opening websites, and searching the web.

---

# 📌 Features

- 🎤 Voice Command Recognition
- 🗣 Text to Speech Response
- ⏰ Tell Current Time
- 🌐 Open Websites (Google, YouTube)
- 🔍 Perform Google Search
- 💻 Open Applications
- 🤖 Custom Voice Commands
- 🧠 Easy to Modify & Extend

---

# 🛠 Technologies Used

- **Python**
- **SpeechRecognition**
- **pyttsx3**
- **PyAudio**
- **datetime**
- **webbrowser**
- **os**

# ⚙ Installation

## Step 1 - Clone Repository

git clone https://github.com/your-username/jarvis-voice-assistant.git

cd jarvis-voice-assistant

---

## Step 2 - Install Dependencies

pip install -r requirements.txt

If PyAudio gives error:

pip install pipwin

pipwin install pyaudio

---

# ▶ Run Project

python jarvis.py
